# Bamboo Mat — Website Prototype

Front-end-only prototype of a redesigned website for [Bamboo Mat](https://bamboo-mat.co.uk), a Nikkei (Peruvian–Japanese) restaurant in Stratford, East London. Built as a client-pitch piece — cinematic, premium, motion-rich.

> **Design prototype — not the live Bamboo Mat website.** No backend, no CMS, no real booking (the reserve flow is a front-end mock; real bookings go via Design My Night).

## Stack

- **Vite** multi-page app — `index`, `menu`, `experiences`, `about`, `gallery`, `contact`
- Vanilla JS + HTML + CSS (no framework)
- **GSAP** with ScrollTrigger + SplitText
- **Lenis** smooth scroll

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
